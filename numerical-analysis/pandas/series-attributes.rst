*****************
Series Attributes
*****************


.. code-block:: python

    import pandas as pd

    s = pd.Series(['a', 'b', 'c'])


Shape
=====
.. code-block:: python

    s.shape
    # (3,)


Number of Dimensions
====================
.. code-block:: python

    s.ndim
    # 1


Index
=====
.. code-block:: python

    s.index
    # RangeIndex(start=0, stop=3, step=1)


Values
======
.. code-block:: python

    s.values
    # array(['a', 'b', 'c'], dtype=object)
